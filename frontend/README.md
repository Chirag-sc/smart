# SMART SIT Frontend (Vite + React)

### Environment Variables

Create a `.env` file in `frontend/` and set:

```
VITE_BACKEND_URL=http://localhost:5001
```

Do not include a trailing slash. Vite exposes variables via `import.meta.env.VITE_*`.

### Development

Install and run:

```
npm install
npm run dev
```
